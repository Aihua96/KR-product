## 维护指引
1. 不直接在自动区域修改：请在对应文档首次出现术语处添加 `<Term name="术语" desc="简短说明" />`。
2. 运行 `npm run generate:glossary` 自动重建表格，列“最后更新”来源于包含该 `<Term>` 标记文件的最近一次 Git 提交时间。
3. 若术语需要删除：先从文档移除 `<Term>` 标记，再运行脚本。
4. 已废弃术语可在描述后添加 `（已废弃）` 保留 2~3 个版本后再移除。
