# 部署指南（KRStorage）

> 描述如何部署 KRStorage（占位）。

## 前置条件
- 已选定后端存储类型（Ceph / NFS / 本地盘等）
- 已准备网络与访问凭据

## 部署步骤（示例）
1. 安装控制组件
2. 配置后端驱动（示例：Ceph Secret / NFS 挂载）
3. 创建存储池与策略
4. 验证容量与性能指标

## 验证
- 创建测试卷/文件共享
- 绑定到上层业务并读写测试
